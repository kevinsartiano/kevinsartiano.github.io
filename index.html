Projects

  • Jira-Slack Notification Bot

    This app combines the Jira API and the Slack API to send custom notifications any time a ticket is assigned to you or your team. I developed this bot to solve an issue that my team had keeping up to date with Jira alerts. It received great feedback and proved itself a big success.

  • Airline Scraper

    My thesis in Computer Engineering aims at finding out whether airlines apply price discrimination when selling tickets. To collect the data, I designed and built a flexible scraper using Python, Selenium and the Amadeus APIs. The research will be published in a scientific journal.

  • Toggl Helper MacOS App

    The goal of this app is to offer an easier and more efficient way of tracking working time in Toggl. The app uses of the package Rumps for the macOS GUI. The data entered in the window is stored in a spreadsheet file and exported to the web platform through the Toggl API.

  • Germany COVID-19 Tracker

    This is a web application that tracks the 7-day cases per 100k inhabitants in Germany using the data sourced from the Robert Koch Institute API. The web application was written using Python, Flask and Folium.

  • German Verbs Android App

    German is hard, but not impossible to learn. To boost my mobile development skills, I created an Android app using Python and Kivy. The app was specifically crafted to help my fellow VHS students (and myself 😅) in learning verb conjugations for upcoming exams.
