Projects

  • Bachelor's Thesis: Airline Scraper

    My bachelor's thesis focussed on finding out whether airlines apply price discrimination when selling tickets. To collect the data, I designed and built a flexible scraper using Python3, Selenium and the Amadeus APIs.

  • German Verbs Android App

    German is hard, but not impossible to learn. To help my fellow VHS students (and myself 😅) in learning German, I created an Android app (using Python3 and Kivy) to practice verb conjugations. I did thid because I believe that we learn better by doing than memorizing.

  • Tic-Tac-Toe Game

    One of my first coding projects was to build a CLI Tic-Tac-Toe game. I have always been fascinated by GUIs, so once I gathered enough experience, I decided to take on the old challenge by developing a GUI for it. I built the game using Python3 and the Tk Interface.
